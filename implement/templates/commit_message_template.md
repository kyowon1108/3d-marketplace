# Commit Message Template

Format:
`<type>(<area>): <intent>`

## Type
- `feat` | `fix` | `refactor` | `docs` | `chore`

## Example
- `docs(ops): define AI delivery workflow and task contract`
- `refactor(db): serialize active output switch by job lock`
- `fix(backend): normalize availability mapping for PREVIEW`

## Validation Note (optional body)
- tests:
- docs synced:
- evidence:
