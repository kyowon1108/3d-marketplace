# Templates Index

이 디렉터리는 AI 작업 위임 시 반복되는 문서 포맷을 표준화한다.

## Available Templates
1. `task_brief_template.md`
2. `commit_message_template.md`
3. `pr_description_template.md`
4. `evidence_report_template.md`
5. `gate_decision_template.md`

## Usage Rules
1. 작업 시작 전 `task_brief_template.md` 채우기
2. 커밋/PR 생성 시 해당 템플릿 사용
3. Gate 판정 전 evidence + gate decision 템플릿 작성
